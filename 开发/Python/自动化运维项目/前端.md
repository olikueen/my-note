## 1. 项目初始化

### 1.1 vue-cli

```shell
# 安装vue-cli
npm install -g @vue/cli

# 查看版本
vue -V
```

## 1.2 创建web项目

```shell
mkdir -pv uric/src
cd uric/src
vue create web
```
